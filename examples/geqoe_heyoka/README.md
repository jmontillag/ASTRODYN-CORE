# GEqOE Heyoka Examples

These scripts cover the new heyoka-backed GEqOE Taylor implementation,
including continuous thrust, propagated mass, and multiple shooting.

Run from the repository root with the project Conda environment, for example:

```bash
conda run -n astrodyn-core-env python examples/geqoe_heyoka/geqoe_taylor_demo.py
conda run -n astrodyn-core-env python examples/geqoe_heyoka/geqoe_taylor_shooting_demo.py
```
