# Averaged GEqOE Theory — First-Order Mixed-Zonal J2–J5

First-order Lie–Deprit averaging of the Generalized Equinoctial Orbital Elements
(GEqOE) under mixed zonal harmonics J2–J5. The theory produces closed-form
secular/long-period mean rates plus an exact short-period map, propagated with
an eccentric-longitude (K) formulation compatible with heyoka's expression DAG.
Validated against Cowell numerics and Brouwer theory across 12 orbital regimes.

## How to run

```bash
conda run -n astrodyn-core-env python docs/geqoe_averaged/scripts/<script>.py
```

To compile the main paper:
```bash
cd docs/geqoe_averaged && latexmk -pdf geqoe_averaged_j2_zonal_note.tex
```

## Dependencies

- **astrodyn_core.geqoe_taylor** — osculating GEqOE propagator (heyoka-based)
- **heyoka** ≥ 7.9.2 — Taylor integrator
- **sympy** — symbolic averaging and coefficient generation
- **numpy**, **matplotlib** — numerics and figures
- **orekit** (optional) — Brouwer comparison in extended validation

## Relationship to astrodyn_core

The scripts import from the osculating propagator (`astrodyn_core.geqoe_taylor`)
for numerical ground truth. The averaged theory is **not yet integrated** as a
module in `src/astrodyn_core/`; it remains a research study in `docs/`.
